# Google Home Bridge add-on repository

Add-on repository containing **Google Home Bridge** — self-hosted Google
Assistant voice control for Home Assistant, replacing the Google part of a
Nabu Casa subscription.

## Installation

1. Settings → Add-ons → Add-on store → ⋮ → Repositories → add this repo URL.
2. Install **Google Home Bridge**, start it, open the Web UI and follow the wizard.

## Add-ons

- **[Google Home Bridge](./google_home_bridge)** — expose entities to Google
  Assistant with the native UI, your own Google project, and no subscription.

Not on Home Assistant OS/Supervised? The bundled custom integration is also
published standalone at
[domenix/google-home-bridge-integration](https://github.com/domenix/google-home-bridge-integration).
