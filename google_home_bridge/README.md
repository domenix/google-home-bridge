# Google Home Bridge

Self-hosted Google Home / Google Assistant voice control for Home Assistant —
a drop-in replacement for the Google part of a Nabu Casa subscription.

- Native per-entity expose toggles (Settings → Voice assistants), same as Cloud
- Live state reporting to the Google Home app via your own HomeGraph service account
- Built-in Cloudflare Tunnel or direct TLS exposure — only the 3 endpoints Google needs are public
- Guided setup wizard with validation (ingress web UI)

See the **Documentation** tab for setup.
